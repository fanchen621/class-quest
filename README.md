# ⚔️ Class Quest · 课堂冒险

> **班级优化大师 × 顶尖精美卡牌游戏** — 把课堂管理变成一场史诗冒险

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ 特性

### 🏠 班级管理（优化大师核心）
- **英雄系统** — 每位学生是独一无二的英雄角色，拥有职业、等级、属性
- **积分体系** — 快捷加分/扣分，支持自定义原因（积极发言、作业优秀、帮助同学等）
- **随机点名** — 炫酷转盘随机选择学生，自动奖励积分
- **智能分组** — 一键随机分组，支持多组模式
- **行为记录** — 完整的积分历史追踪
- **批量导入** — 支持批量添加学生（姓名+职业）

### 🃏 卡牌游戏（核心玩法）
- **30+ 精美卡牌** — 攻击、防御、法术、增益、传说五大类型
- **四档稀有度** — 普通 ⭐ / 稀有 ⭐⭐ / 史诗 ⭐⭐⭐ / 传说 ⭐⭐⭐⭐
- **卡牌图鉴** — 收集系统，未获得卡牌显示剪影
- **卡牌详情** — 精美卡面展示，含属性、描述、效果

### ✨ 召唤系统（Gacha）
- **单抽/十连** — 消耗💎召唤卡牌
- **保底机制** — 90抽必出传说，公平公正
- **史诗动画** — 卡牌翻转揭晓特效
- **卡牌分配** — 自动将卡牌分配给随机学生

### ⚔️ 对战系统
- **1v1 决斗** — 两位英雄的史诗对决
- **小组对抗** — 团队荣誉之战
- **BOSS 挑战** — 全班协力击退暗影巨龙
- **回合制战斗** — 法力值、手牌、技能释放
- **AI 对手** — 智能敌人自动出牌

### 🏆 荣耀殿堂
- **积分排行** — 谁是课堂最强王者
- **卡牌排行** — 收藏家的较量
- **胜场排行** — 战斗大师排名
- **成就排行** — 成就达人榜

### 🎖️ 成就系统
- 🩸 第一滴血 — 首次对战胜利
- 📦 收藏家 — 收集10张卡牌
- 💰 小富翁 — 累计500金币
- ⚔️ 五连胜 — 连赢5场
- 🌟 欧皇降临 — 抽到传说卡
- 💯 百分达人 — 累计100积分
- ...更多成就等你解锁！

### 📜 每日任务
- 随机刷新4个每日任务
- 完成任务获得金币和💎奖励

## 🚀 使用方式

直接打开 `index.html` 即可使用，无需服务器，数据自动保存在浏览器 localStorage。

```bash
# 本地运行
cd class-quest
python3 -m http.server 8080
# 打开 http://localhost:8080
```

## 🎨 技术栈

- **纯前端** — HTML + CSS + JavaScript，零依赖
- **粒子背景** — Canvas 动态粒子系统
- **3D 卡牌** — CSS Transform 实现卡牌翻转效果
- **响应式** — 完美适配桌面端和移动端
- **数据持久化** — localStorage 本地存储
- **暗色主题** — 护眼深色配色，紫金色调
